var

  1. it can be updated and redeclared, but it creates unnecessary space in the memory location
  2. it is not a fixed value and can be updated but creates different memory location for storing data
  3. Example: we declare var in the following way:var a=2; var a=2; -declaration not a fixed value but different memory location (unnecessary space is created) var a=3; -value reassign possible but on different memory location no errors

    let

  1. it can be updated and redeclared, but it does not create unnecessary space in the memory location
  2. it is not a fixed value and can be updated but on the same memory location is used
  3. Example: we declare let in the following way:let a=2; let a=2; -declaration not a fixed value but same memory location (no unnecessary space is created) let a=3; -value reassign possible but on same memory location no errors

    const

  1. it cannot be updated and redeclared, but donot creates unnecessary space in the memory location
  2. it is a fixed value and cannot be changed once its been declared
  3. Example: we declare const in the following way: const a=2; -declaration fixed value (no unnecessary space is created in memory location) const a=3; -not possible result in error.
